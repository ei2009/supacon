SuperCon 2021 Judge

Before using this, you have to copy this system directory to your home directory; like:
 $ cp -r /home/pub/syoribu/SuperCon2021Judge $HOME

Usage:
1) copy your source code; like:
 $ cp /path/to/Main.cpp /path/to/SuperCon2021Judge/Main.cpp

  IMPORTANT NOTICE: You should make filename "Main.cpp"

2) SSH into cricket:
 $ ssh cricket

3) run judge system:
 $ bash ./judge.sh
